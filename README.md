# Telecom X – Predicción de Cancelación de Clientes

## Descripción

Este proyecto tiene como objetivo desarrollar modelos de machine learning capaces de predecir la cancelación de clientes (churn) en la empresa Telecom X.

El análisis parte de un dataset previamente tratado y utiliza técnicas de análisis exploratorio, preparación de datos y modelado predictivo.

## Objetivos

- Preparar los datos para el modelado
- Analizar correlaciones entre variables
- Construir modelos de clasificación
- Evaluar el rendimiento del modelo
- Identificar los factores que influyen en la cancelación

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Modelos utilizados

- Regresión Logística
- Árbol de Decisión

## Resultados

Los modelos permitieron identificar variables importantes asociadas a la cancelación de clientes, como:

- Tipo de contrato
- Tiempo de permanencia
- Cargos mensuales
- Método de pago

Estos resultados pueden ayudar a la empresa a desarrollar estrategias de retención más efectivas.
